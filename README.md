# English

## About

Abricot.co is a collaborative SaaS project management application developed with Next.js and React as part of the OpenClassrooms project:

**“Développez un SaaS de gestion de tâches”**

The application allows users to:

- manage projects
- manage contributors
- create and assign tasks
- track task status
- collaborate through comments
- visualize tasks with list and kanban views

## Installation

### Frontend

```bash
npm install
npm run dev
```

Production:

```bash
npm run build
npm start
```

Frontend runs on:

```text
http://localhost:3000
```

### Backend

```bash
npm install
npm run dev
```

Backend runs on:

```text
http://localhost:8000
```
