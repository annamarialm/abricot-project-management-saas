# English

## About

Abricot.co is a collaborative SaaS project management application built with Next.js and React as part of the OpenClassrooms project:

**“Développez un SaaS de gestion de tâches”**

This frontend includes:

- authentication
- dashboard views
- kanban board
- project management
- task management
- comments system
- permissions system

## Features

- User authentication
- Protected routes
- Project creation/editing
- Contributor management
- Task creation/editing/deletion
- Task comments
- Dashboard list view
- Dashboard kanban view
- Role/permission system

## Installation

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm start
```

Frontend runs on:

```text
http://localhost:3000
```
